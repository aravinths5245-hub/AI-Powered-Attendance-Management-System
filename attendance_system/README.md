# AI Powered Attendance Management System

A complete Flask-based attendance management system with modern UI, authentication, student management, attendance tracking, and AI-inspired face-recognition workflow.

## Features
- Secure login and registration
- Admin dashboard with summary cards and charts
- Student, faculty, department, course, and subject management
- Attendance marking and history
- Responsive premium UI with dark/light mode support
- AI-ready face data storage and recognition workflow

## Setup
1. Create a virtual environment
   - `python -m venv venv`
   - `venv\Scripts\activate`
2. Install dependencies
   - `pip install -r requirements.txt`
3. Run the app
   - `python app.py`
4. Open http://127.0.0.1:5000
